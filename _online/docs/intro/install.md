## **Установка Симплайт 5**

Для установки Симплайт 5 необходимо перейти в директорию, в которую был загружен установочный файл **setup_simplight_5.0.x.x** (точное название может варьироваться в зависимости от версии), и запустить его на выполнение с правами администратора.

<img alt="" height="236" name="simp_install_01_.png" src="https://ucarecdn.com/4fb54701-8edc-4460-b2b4-6f57041f9f44/" style="border:1px solid #cccccc" width="607" />

<img alt="" height="145" name="2025-06-04_08-53.png" src="https://ucarecdn.com/e14e8a89-21e8-425b-95c8-341cea89d436/" style="border:1px solid #cccccc" width="361" />

Чтобы продолжить процесс установки, необходимо ознакомиться с условиями лицензионного соглашения и принять их, выбрав соответствующий флаг.

<img alt="" height="402" name="2025-06-04_08-55.png" src="https://ucarecdn.com/f7f17d43-f73b-453b-b43f-4e9744d2a5e8/" style="border:1px solid #cccccc" width="500" />

На следующем этапе пользователю предлагается выбрать каталог, в котором среда разработки будет сохранять файлы проектов, такие как списки и свойства тегов, мнемосхемы, скрипты и базы данных.


!!! warning "Внимание"
    Рекомендуется заранее выбрать корректный каталог для хранения данных, поскольку после установки Симплайт5 изменить расположение каталога хранения будет невозможно. Чтобы изменить расположение файлов проекта, необходимо полностью удалить программное обеспечение и при установке указать правильный путь

<img alt="" height="377" name="2025-06-04_08-55_1.png" src="https://ucarecdn.com/c9a3375e-4267-40f1-8b1d-7a3ea011bef4/" style="border:1px solid #cccccc" width="500" />

В окне **Выбор компонентов** необходимо отметить компоненты, которые понадобятся для разработки проектов на ПК. В зависимости от необходимого функционала для разработки и запуска ПО можно рассмотреть следующие комбинации.

=== "Рабочая станция разработчика"
	Среда разработки  
	Сервер  
	HMI клиент

=== "Серверная машина"
	Сервер


=== "Клиент"
	HMI-клиент

!!! info "Информация"
	Если планируется устанавливать сервер на ПК с Linux, то следует также выбрать флаг **Файлы для Linux**. Файлы для установки сервера на ОС Linux будут помещены в папку **C:\Users\Public\Documents\SimpLight_5.0\Linux files**. Данные файлы копируются на Linux любым удобным способом и пользователь может самостоятельно выполнить стандартную установку .deb пакетов в ОС.


При выборе среда установки автоматически активирует флаг **Установить сервер как службу** &mdash; это предпочтительный вариант работы сервера.  
Если ПК выступает только в роли клиента, то можно выбрать  флаг **HMI клиент** &mdash; данная настройка позволит использовать на ПК независимый клиент наряду с web-клиентом.

<img alt="" height="380" name="2025-06-04_08-59.png" src="https://ucarecdn.com/86deb4dc-5c9c-4bf2-a7ed-3d34baf5df54/" style="border:1px solid #cccccc" width="500" />

На этом этапе предлагается выбрать путь к системным файлам ПО.

<img alt="" height="376" name="2025-06-04_09-00_1.png" src="https://ucarecdn.com/d0ecad2e-0865-4c55-8156-7fe2686e612c/" style="border:1px solid #cccccc" width="500" />

Далее необходимо выбрать название папки для хранения ярлыков системных файлов меню **Пуск**.

<img alt="" height="381" name="2025-06-04_09-01.png" src="https://ucarecdn.com/23a1b4bf-dc82-4809-bfd3-1b549eed62ff/" style="border:1px solid #cccccc" width="500" />

Перед началом установки система указывает путь к каталогам, которые были выбраны пользователем на предыдущих этапах. Если необходимо внести изменения, то следует вернуться к предыдущим шагам, нажав на кнопку **Назад** и выполнить исправления.

<img alt="" height="379" name="2025-06-04_09-01_1.png" src="https://ucarecdn.com/28b528ec-64c4-4f08-b15a-2da2c9dfec32/" style="border:1px solid #cccccc" width="500" />

Далее откроется окно с прогрессом установки ПО.

<img alt="" height="376" name="simp_install_08.png" src="https://ucarecdn.com/a8422e69-c8c2-40ec-9cf6-8a8ef1c1e821/" style="border:1px solid #cccccc" width="500" />

Для завершения работы мастера установки Симплайт 5 следует нажать кнопку **Завершить**.

<img alt="" height="378" name="simp_install_09.png" src="https://ucarecdn.com/0d713b4c-6acb-47bb-8fa1-5fb547e02c24/" style="border:1px solid #cccccc" width="500" />
